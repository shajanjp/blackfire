# BlackFire
Easy to use, maintainable, modular MEAN boilerplate.

## Download
```
git clone https://github.com/shajanjp/blackfire.git my-app
cd my-app
```

## Install Dependencies
```
yarn install
```
or
```
npm install
```

## Usage
```
./make module car cars Cars
```

## Run
```
yarn start
```
or
```
npm start
```
