# blackfire
Easy to use, maintainable MEAN boilerplate.

## Installation
```
git clone https://github.com/shajanjp/blackfire.git my-app
cd my-app
```

## Dependancy Installation
```
yarn install
```
or
```
npm install
```

## Usage
```
./make module car cars Cars
```

## Execution
```
yarn start
```
or
```
npm start
```
